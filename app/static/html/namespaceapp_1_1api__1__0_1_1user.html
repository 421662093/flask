def app.api_1_0.user.change_phone ( ) 
更改用户手机号（帐号）

URL:/user/changephone
格式
    JSON
POST 参数:
    username -- 帐号 (必填)
    code -- 手机验证码 (必填)
返回值
    {'ret':1,'username':'帐号'} 发送成功

    -2 帐号已存在

    -4 手机号格式错误
    -5 系统异常
    -6 帐号更新异常
